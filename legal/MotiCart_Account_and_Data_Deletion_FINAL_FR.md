# Suppression du compte et suppression des données — MotiCart

**Dernière mise à jour :** 5 juillet 2026

Cette page explique comment les utilisateurs peuvent supprimer leur compte MotiCart et demander la suppression des données liées à l'application.

**Nom de l'application :** MotiCart  
**Développeur :** MotikaBot  
**Nom du package :** com.motikabot.moticart  
**E-mail de contact :** motikabot@gmail.com

## 1. Supprimer votre compte dans l'application

Si vous êtes connecté à MotiCart, vous pouvez supprimer votre compte directement dans l'application :

1. Ouvrez MotiCart.
2. Allez dans **Paramètres**.
3. Choisissez **Supprimer toutes les données**.
4. Confirmez la demande de suppression.

Vous devez être en ligne et connecté pour que la suppression du compte dans l'application soit effectuée.

## 2. Demander la suppression par e-mail

Si vous ne pouvez pas accéder à l'application, vous pouvez demander la suppression du compte par e-mail.

Envoyez votre demande à :

**motikabot@gmail.com**

Veuillez inclure :

- **Objet :** Demande de suppression du compte MotiCart
- L'adresse e-mail utilisée pour vous connecter à MotiCart.
- Un court message indiquant que vous souhaitez que votre compte MotiCart et les données d'application associées soient supprimés.

Pour des raisons de sécurité, nous pouvons devoir vérifier que vous êtes le propriétaire du compte avant de traiter la demande.

Les demandes de suppression doivent normalement être envoyées depuis la même adresse e-mail que celle utilisée pour vous connecter à MotiCart. Si vous n'avez plus accès à cette adresse e-mail, veuillez expliquer la situation dans votre demande. Si nous ne pouvons pas raisonnablement vérifier que vous êtes le propriétaire du compte, il se peut que nous ne puissions pas traiter la demande de suppression par e-mail.

Les demandes de suppression par e-mail peuvent supprimer ou retirer les données cloud liées au compte de notre backend. Elles ne peuvent pas supprimer directement les données locales stockées uniquement sur votre appareil. Pour supprimer les données locales de l'appareil, vous pouvez supprimer les listes dans l'application, effacer les données de l'application dans les paramètres Android ou désinstaller l'application.

## 3. Ce qui est supprimé

Lorsque vous supprimez votre compte MotiCart, nous supprimons ou retirons les données liées au compte de notre backend d'application, notamment :

- votre profil de compte MotiCart ;
- votre adresse e-mail et votre nom d'affichage de notre backend d'application ;
- votre identifiant utilisateur MotiCart de notre backend d'application ;
- les listes partagées ou familiales dont vous êtes propriétaire ;
- les articles de listes partagées, les codes d'invitation, l'historique d'archive et les enregistrements associés pour les groupes dont vous êtes propriétaire ;
- vos références d'adhésion dans les groupes partagés appartenant à d'autres utilisateurs.

Lorsque la suppression du compte est effectuée depuis l'application, MotiCart est également conçue pour vous déconnecter localement et effacer les données locales de l'application stockées par MotiCart sur cet appareil, telles que la base de données locale, les préférences de l'application, les fichiers mis en cache et le stockage local de l'application.

## 4. Groupes partagés appartenant à d'autres utilisateurs

Si vous avez rejoint une liste partagée ou familiale appartenant à un autre utilisateur, la suppression de votre compte ne supprime pas automatiquement le groupe partagé de ce propriétaire.

Dans ce cas, MotiCart supprime vos références d'adhésion et votre identité de compte de ce groupe lorsque cela est possible, tout en conservant le groupe pour les membres restants.

## 5. Utilisation locale uniquement sans compte

Si vous utilisez MotiCart sans vous connecter, vos listes de courses locales sont stockées sur votre appareil.

Comme aucun compte cloud MotiCart n'existe dans ce cas, il n'y a pas de compte en ligne à supprimer. Vous pouvez supprimer les données uniquement locales en supprimant les listes dans l'application, en effaçant les données de l'application dans les paramètres Android ou en désinstallant l'application.

## 6. Demander la suppression d'autres données d'application

Si vous souhaitez demander la suppression de données liées à l'application sans supprimer l'intégralité de votre compte, contactez-nous à :

**motikabot@gmail.com**

Veuillez inclure l'adresse e-mail utilisée pour vous connecter à MotiCart et décrire les données que vous souhaitez supprimer.

Certaines données peuvent être nécessaires au fonctionnement de votre compte ou des fonctionnalités de listes partagées. Si la suppression de données spécifiques empêchait une fonctionnalité de fonctionner, nous pouvons vous expliquer les options disponibles, comme supprimer votre compte ou quitter un groupe partagé.

## 7. Abonnement et enregistrements de facturation

La suppression de votre compte MotiCart n'annule pas automatiquement un abonnement Google Play actif.

Si vous avez un abonnement Plus ou Family actif, vous devez l'annuler dans les paramètres de votre compte Google Play afin d'arrêter les futurs renouvellements.

La désinstallation de MotiCart n'annule pas automatiquement votre abonnement.

MotiCart ne stocke pas votre numéro de carte de crédit, vos coordonnées bancaires ou les informations complètes relatives à votre moyen de paiement. Les paiements sont traités par Google Play.

Certains enregistrements de facturation, de transaction, fiscaux, de prévention de la fraude, de remboursement, de litige ou de conformité légale peuvent rester auprès de Google Play, RevenueCat ou d'autres prestataires de services liés aux paiements, conformément à leurs propres exigences légales et opérationnelles.

## 8. Données pouvant être conservées pendant une durée limitée

Certains enregistrements techniques peuvent rester pendant une période limitée lorsque cela est nécessaire pour la sécurité, la prévention de la fraude, les diagnostics de plantage, la conformité légale, la facturation, le traitement des litiges ou l'intégrité des sauvegardes.

Par exemple, les rapports de plantage et les enregistrements de diagnostic peuvent être conservés par Firebase Crashlytics pendant une période limitée conformément à ses pratiques de conservation.

Nous n'envoyons pas intentionnellement le contenu des listes de courses, les adresses e-mail, les jetons d'authentification ou les détails de cartes de paiement dans les rapports de plantage.

## 9. Délai de traitement

Nous nous efforçons de traiter les demandes de suppression dès que raisonnablement possible après leur réception et leur vérification.

Certaines demandes peuvent prendre plus de temps si une vérification est nécessaire, si la demande n'est pas claire ou si certaines données doivent être conservées pour des raisons juridiques, de sécurité, de facturation, de prévention de la fraude, de résolution des litiges ou de conformité aux plateformes.
